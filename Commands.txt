==BASICO==
git init => Cria os arquivos necessarios para configuracao

git add . => Adiciona todos os arquivos 

git commit -m "mensagem" => faz o commit dos pacotes

git push => sobe para o github


==VERSOES==
git reflog => Ver todas as versoes do codigo


git reset --hard IDDAVERSAODESEJADA


==BRANCH==
git branch => ver as branchs

git branch nomedabranch => Criar uma branch nova

git checkout novabranch master => Criacao de uma nova branch baseada em outra branch

git checkout nomedabranch => Serve para selecionar a branch desejada



==PROCESSO DE MERGE==

Atualiza a branch master trazendo o codigo de outra branch para ele

Entre na branch que deseja atualizar
        git checkout master

Seleciona o comando com a branch que deseja subir para a branch que está atualmente
        git merge staging

==PUXANDO ATUALIZAÇOES==
git pull => Puxa as atualizacoes do github
